# 快捷键

通用快捷键

|win                    |mac      |功能 |
|:----------------------|:--------|:----|
|Ctrl + O               |⌃ O      |打开|
|Ctrl + E               |⌃ E      |执行|
|Shift + R              |⌃ R      |打开项目文件夹|
|Ctrl + S               |⌃ S      |保存|
|Ctrl + Shift + S       |⌃ ⇧ S    |另存为|
|Ctrl + Shift + H       |⌃ ⇧ H    |隐藏/显示编辑区|
|Ctrl + P               |⌃ P      |发布项目|
|Ctrl + Shift + F       |⌃ ⇧ F    |查找和创建拼图|
|Ctrl + I               |⌃ I      |打开场景查看器|
|Shift + D              |⌃ D      |复制选中拼图|
|Ctrl + D               |⌃ D      |启用/禁用选中拼图|
|Ctrl + Z               |⌃ Z      |撤消|
|Ctrl + Shift + Z       |⌃ ⇧ Z    |重做|
|Alt + Shift + F        |⌃ ⇧ F    |整理拼图|
|Backspace / Delete     |Delete   |删除拼图|

创建拼图快捷键

|win             |mac  |拼图                    |描述        |
|:---------------|:----|:----------------------|:-----------|
|Shift + T       |⇧ T  | text                  |创建文本拼图|
|Shift + N       |⇧ N  | math_number           |创建数字拼图|
|Shift + B       |⇧ B  | logic_boolean         |创建布尔拼图|
|Shift + A       |⇧ A  | array_create          |创建数组拼图|
|Alt + Shift + A |⌥ ⇧ A| array_create_element  |创建数组元素拼图|
|Shift + O       |⇧ O  | object_create         |创建对象拼图|
|Alt + Shift + O |⌥ ⇧ O| object_key_value      |创建对象键值对拼图|
|Shift + V       |⇧ V  | variables_set_dynamic |创建变量set拼图|
|Alt + Shift + V |⌥ ⇧ V| variables_get_dynamic |创建变量get拼图|
|Shift + F       |⇧ F  | procedures_defnoreturn|创建函数拼图|
|Shift + L       |⇧ L  | console_log           |创建控制台打印拼图|
|Shift + C       |⇧ C  | advanced_comment      |创建注释拼图|
|Alt + Shift + C |⌥ ⇧ C| advanced_connect      |创建连接拼图|

特殊拼图快捷键

- Tab - 作用于`exec_tab`拼图时，跳转到相应选项卡
- Shift + Tab - 从自定义选项卡返回`main`选项卡
